ITNP-Chat
Hybride Chat-Applikation
Vielfältig verwendbar
Sende oder Empfange Nachrichten, Bilder & Videos, Dateien, Orte, Sprachaufnahmen, etc., erstelle Gruppen oder ordne dich selber einer vordefinierten Kategorie zu.
Sicheres Chatten
Alle gesendeten Nachrichten werden durch ein asymmetrisches Kryptographieverfahren verschlüsselt und nach einer bestimmten Zeit für immer vom Server gelöscht.
Lokaler Server
Durch bestimmte Dynamik ist es möglich einen eigenen Server lokal zu installieren. Somit gelangen verschlüsselte Nachrichten nie in falsche Hände.
Erweiterte Dynamik
Die Applikation enthält dynamische Variablen welche bestimmte Pfade zu Bildern, Farben, Schriftarten oder Größen enthält. Gestalte Sie deinen Wünschen nach!
Echtzeitige Datenübertragung
Echtzeitig werden die Daten zum Server und weiter zum Empfänger gesendet. Das NODE.JS-Modul SOCKET.IO ermöglicht eine professionelle Datenübertragung.
